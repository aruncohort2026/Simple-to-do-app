# 📝 Simple To-Do App

A minimal and responsive To-Do List application built using **HTML, CSS, and Vanilla JavaScript**.

## 🚀 Features
- Add, delete, and mark tasks as completed  
- Real-time task statistics (Total, Completed, Remaining)  
- Clear completed tasks  
- Data persistence using LocalStorage  
- Responsive design  

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  

## ▶️ How to Run
1. Download or clone the repository  
2. Open `index.html` in your browser  

## 💾 Storage
Tasks are stored in the browser using **LocalStorage**, so data persists after refresh.
